My projects with PCB Design

STM32F072 Board

After I took a microprocessor class, I became fascinated with microcontrollers.

So I thought, "Why not make my own custom microcontroller PCB?" and so I did!

This specific PCB you're looking at runs an ARM-based STM32F072 microcontroller.
It contains the actual microcontroller itself, a USB-to-UART converter, a dedicated debugging port labeled "SWD",
and 4 ports where you can plug in "breakout" boards.

ICM-20948 Breakout Board

An MCU, admittedly, is not that useful on its own without some kind of peripheral device(s).

IS61LV256AL-10TLI Breakout Board

While the STM32 has memory, it's very limited. It only has 16 kilo-bytes of RAM and 128 kilo-bytes of flash memory.

So, I decided to expand it myself by using a breakout board! Now, we have plenty of memory to work with!
