My projects with Verilog

PWM Controller

This design works by having an 8 bit counter switch between "1" and "0" via a 2 input mux. If you need to change the frequency at which the PWM outputs, you can latch a new value into the counter and reset the device for it to take effect.

Up/Down Counter

This is a very simple up/down counter with preload capability.

You can preload a value into the counter, and it can count up to that value starting at zero, or it can count down from that value down to zero. Depending on what value the direction input is set to.

n-bit register

This is a register that you can specify the width of.

It can be 64 bits wide, 32 bits wide, 16 bits, 8 bits, 4 bits, 2 bits, 1 bit, but not half bit!
